# Tai Ray Music - Official Website

This is the official website for Tai Ray, built with [Next.js](https://nextjs.org).

## Features

- Beautiful purple-themed design
- Responsive layout for all devices
- Links to YouTube and Facebook
- Newsletter signup
- Modern typography and animations

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
